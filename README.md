# hdrelhaj.com

Personal portfolio website for **Abdelrehman Elhaj** — Systems & Cloud Engineer based in Riyadh, Saudi Arabia.

Live at [hdrelhaj.com](https://hdrelhaj.com)

## Stack

- Single `index.html` — no frameworks, no build tools
- Deployed on **Cloudflare Pages** via GitHub
- Fonts: Cormorant Garamond · DM Mono · Outfit (EN) / Amiri · Tajawal (AR)

## Features

- Bilingual **English / Arabic** with full RTL support — toggle persists via `localStorage`
- Hero with photo, stats, and amber editorial aesthetic
- Projects, Skills, Experience, Certifications sections
- Mobile hamburger nav with scroll spy
- Contact form powered by [Web3Forms](https://web3forms.com)
- Favicon + Open Graph / Twitter card meta tags with custom `og.png`

## Deployment

Push to `main` → Cloudflare Pages auto-deploys.

```
git add .
git commit -m "your message"
git push origin main
```

## Local Development

Open `index.html` directly in a browser — no server needed.
